## LAB 2 - Diagnóstico de Conectividade (Windows 11 Home)

## Objetivo

Simular e diagnosticar falhas comuns de conectividade em rede doméstica utilizando Windows 11 Home e ferramentas nativas do sistema.

## Ferramentas Utilizadas
- ipconfig
- ping
- tracert
- nslokuup
- Configurações de Rede Windows 11

## Competências Desenvolvidas
- Análise de configuração IP
- Diagnóstico de falhas de DNS
- Identificação de falhas entre host e gateway
- Interpretação de rota com tracert
- Raciocínio estruturado para suporte técnico

## O repositório está organizado em:
- Documentação
- Cenários Simulados
- Evidências (prints/imagens)
- Relarório Técnico final

## Aplicação Prática
Laboratório desenvolvido com foco em preparação para atuação como Analista de Suporte / Help Desk Júnior ou Estagiário em Suporte / Help Desk
