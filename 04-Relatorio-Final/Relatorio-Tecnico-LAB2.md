# Relatório Técnico - LAB 2

## Ambiente

- Sistema Operacional: Windows 11 Home
- Tipo de Rede: Doméstica (Wi-Fi)
- Equipamento: Notebook pessoal

## Objetivo

Desenvolver a capacidade de diagnóstico estruturado para falhas de conectividade.

## Cenários executados

1. Perda total de conexão
2. Falha de DNS
3. Resolução por IP mas não por nome
4. Falha de navegação com Wi-Fi conectado

## Ferramentas utilizadas

- ipconfig
- ping
- nslookup
- tracert

## Principais aprendizados

- Diagnóstico deve seguir ordem lógica
- DNS é causa comum de falhas
- Gateway é ponto crítico de teste
- Tracert auxilia na identificação de falha externa

## Conclusão

O laboratório permitiu desenvolver raciocínio técnico aplicado à realidade de suporte de primeiro nível, utilizando apenas recursos nativos do Windows 11 Home.
