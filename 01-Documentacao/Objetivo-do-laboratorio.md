# Objetivo do Laboratório

Este laboratório tem como finalidade desenvolver raciocínio técnico para diagnóstico de falhas de conectividade em ambiente doméstico.

O foco é identificar em qual ponto da comunicação ocorre a falha:
Computador -> Rede Local -> Roteador -> Internet -> DNS -> Serviço

O laboratório foi executado em:
- Windows 11 Home;
- Rede Wi-Fi residencial;
- Roteador padrão de operadora.
